# KLOT

A full stack e-commerce website for a luxury streetwear brand. Built with React and FastAPI.

## Tech Stack

**Frontend** — React, Vite, Framer Motion, React Router, Lucide React

**Backend** — FastAPI, SQLite, PyJWT

## Environment Variables

Create `klot-backend/.env` and `klot-website/.env` using the provided `.env.example` files in each folder.
